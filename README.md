# カウントバトル

ブラウザで動く、カード要素つきの「サーティー(数を言い合って上限に到達したら負け)」ゲームです。
HTML / CSS / JavaScript のみで作られており、ビルド不要で `index.html` を Edge か Chrome で開くだけで遊べます。

## 遊び方

1. `index.html` を Chrome または Edge で開く(ダブルクリックでもOK)。
2. セットアップ画面で人数・上限の数・1ターンで進められる最大数・初期手札の枚数を決めてゲーム開始。
3. 自分の番になったら「自分の番を始める」を押す(同じ端末で対戦する際、他のプレイヤーに手札を見せないための目隠し画面です)。
4. 手札のカードを使ってから(通常カードは1ターン1枚まで。「速攻」付きは何枚でも)、1〜最大値の中で数を選んでカウントを進める。
5. 上限の数に到達してしまったら、その時点でゲーム終了。到達した1人が負け、それ以外の全員が勝者。

## ルールの解釈について(補足)

要望文だけでは一意に決まらない部分があったため、以下のように具体化しました。変更したい場合は `game.js` の該当箇所を調整してください。

- **カウントの進め方**: 現在のカウントに対して 1〜K(設定した最大カウント数)の好きな数だけ進めます。古典的な「31ゲーム」と同じ形式です。
- **敗北条件**: 自分の番で上限の数に到達(またはそれ以上)させてしまった時点でゲーム終了。敗者はその1人だけで、それ以外のプレイヤー全員が勝者になります(「まもり」カードを使っていれば1回だけ回避できます)。カードの効果で直接カウントを動かした場合(タイムトリックなど)は敗北判定を行いません。以降そのカウントのまま次のプレイヤーの番になります。
- **手札**: 最初に1回だけ配られます(追加ドローは無し)。ただし「ドロー」カードを使うと山札から引けます。
- **カードの使用回数**: 通常のカードは1ターンに1枚しか使えません。「速攻」の副効果を持つカードだけは、この制限を無視して何枚でも使えます。制限を超えて通常カードを使おうとした場合は不発になり、手札には残ります。
- **墓地**: 使用されたカード(cha・enc含む)はすべて墓地に置かれます。「墓荒らし」「コピー」など墓地や直前のカードを参照する効果があります。

### 副効果・分類

- **速攻**: 1ターン1枚の制限を無視して何度でも使えます。
- **cha**(chainの略): 自分の手札クリックでは使えず、直前に場に出たカード(host)に対して「cha発動チャンス」でのみ割り込んで使えます。場札はスタック(後入れ先出し)として扱われ、cha付きカード自身に対してもさらに別のcha付きカードで割り込めます(例: 「chaカード禁止」を「chaカード禁止」で打ち消せば、二重否定で元のカードを無理やり通せます)。手番順に1人ずつ確認し、cha持ちのカードを1枚も持っていないプレイヤーは目隠し画面を出さず黙ってスキップします。持っているカードは、今の場札に割り込めない場合でも一覧には表示されますが、押せない状態(グレーアウト)になります。
- **enc**(enchantの略): 自分の手札クリックでは使えず、他のカードを使う際の「同時に使うカードを選ぶ」ステップでのみ添付できます。**1枚のカードに複数のencを同時添付可能**です。数値効果(`magnitude`)を持つカードにのみ効果があり、それ以外のカードには影響しません(消費はされます)。
- **攻撃**: 対象のプレイヤーを1人選んで不利な効果を与えるカードです。「reflection」は攻撃を受けた直後にcha発動チャンスで使うかどうかを選べ、使えば効果が発動者自身に跳ね返ります。

### カード一覧

| カード | 分類/副効果 | 効果 |
| --- | --- | --- |
| 倍プッシュ | - | 上限の数を2倍にする |
| カウントブースト | - | このターンだけ、進められる数の最大値が+2される |
| ダッシュ | 速攻 | このターンだけ、進められる数の最大値が+1される。何枚でも重ねて使える |
| タイムトリック | - | 現在のカウントを+5か-5する(0未満にはならない、選択式) |
| スキップ | 速攻 | 次のプレイヤーの番を1回とばす |
| ぎゃくしゅう | - | 手番が進む方向を逆にする |
| しばりナンバー | - | 次のプレイヤーは、自分のターンで1つしか進められなくなる |
| ドロー | - | 山札からカードを2枚引く |
| まもり | - | 上限に到達しても1回だけ負けを回避する(発動時カウントは0に戻る) |
| パス | - | カウントを進めずに自分のターンを終える(例: 上限60・現在59など、進めると必ず負ける時に有効) |
| 強くてニューゲーム | - | カウントを0に戻し、手札をすべて山札に戻してシャッフルし、開始時の手札枚数+2枚を引き直す。さらに自分だけ最大カウント数が2倍になる |
| choose | 速攻 | 山札の上から5枚を見て、その中から好きな1枚を引く(残り4枚は山札に戻してシャッフル) |
| steal | 攻撃 | 対象を1人選び、その人の手札からランダムに1枚奪う |
| 墓荒らし | - | 墓地からカードをランダムに1枚引く |
| コピー | - | 直前に使われたカードの効果をもう一度発動する(コピー自身や、cha・encのカードはコピーできない) |
| chaカード禁止 | cha | 直前に場に出たカードの効果を打ち消す。打ち消しに打ち消しを使えば、無理やり通すこともできる |
| chaコピー | cha | 直前に場に出たカードの効果を自分にもコピーする(cha付きカードにも使える。例: 「chaカード禁止」をコピーして自分もその対象を打ち消す) |
| reflection | cha | 自分が「攻撃」カードの対象になった直後にだけ使え、その効果を発動者に跳ね返す |
| up grade | enc | 同時に使ったカードの数値効果を変化させる。1枚添付で1.5倍、2枚以上は添付した枚数倍(小数点切り捨て) |
| speed up | enc | 同時に使ったカードに「速攻」を与え、このターンの使用回数制限を無視できるようにする |

カードの種類・枚数・効果は `game.js` の `CARD_TYPES` 配列にまとまっているので、増やしたり数値を調整したりしやすい構成にしています。分類は `category`、副効果は `sub` 配列、数値効果は `magnitude` フィールドで表現しています。

**新しいカードを追加したい場合は [`CARD_GUIDE.md`](./CARD_GUIDE.md) を参照してください。** よくあるパターンをコピペで使えるテンプレートとしてまとめてあります。

### 使えないカードの表示について

自分の手札のうち、今この瞬間に使えないカード(cha・enc付きカード、1ターン1枚の制限を超えた通常カードなど)は、押せない状態(グレーアウト+理由付きツールチップ)で表示され、誤って押しても何も起きない、ということがないようになっています。
